# Data types in Rust
---

### Integer types

| Length | Signed | Unsigned |
|--------|--------|----------|
| 8-bit  | i8     | u8       |
| 16-bit | i16    | u16      |
| 32-bit | i32    | u32      |
| 64-bit | i64    | u64      |
| arch   | isize  | usize    |

### Integer literals

| Number literals | Example |
|-----------------|---------|
| Decimal         | 98_222  |
| Hex             | 0xff    |
| Octal           | 0o77    |
| Binary          | 0b1111_0000 |
| Byte (u8 only)  | b'A'    |

### Floating-point types

| Length | Type |
|--------|------|
| 32-bit | f32  |
| 64-bit | f64  |
